## 3. What I built or tried
```python
import numpy as np

a = np.array([1.0, 2.0, 3.0])
b = np.array([0.5, -1.0, 2.0])
print("dot =", np.dot(a, b))
print("shapes", a.shape, b.shape)

# Mini matrix multiply demo
X = np.array([[1.0, 2.0], [3.0, 4.0]])  # 2 examples × 2 features
w = np.array([0.5, -1.0])
print("X @ w =", X @ w)
```
