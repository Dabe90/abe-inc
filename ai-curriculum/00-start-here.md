## 2. Tools to install (one evening)

### Required
- [ ] [Python 3.11+](https://www.python.org/downloads/)
- [ ] [VS Code](https://code.visualstudio.com/) or Cursor (you already have Cursor)
- [ ] [Git](https://git-scm.com/)
- [ ] GitHub account
- [ ] Free [Google Colab](https://colab.research.google.com/) account (GPU)

### Recommended
- [ ] [Hugging Face](https://huggingface.co/) account
- [ ] [Kaggle](https://www.kaggle.com/) account (datasets + free GPU)
- [ ] Notion or Obsidian (optional) — or just use `my-notes/` here

### One-time setup commands

```bash
python -m venv .venv
# Windows PowerShell:
.\.venv\Scripts\Activate.ps1
pip install numpy matplotlib jupyter scikit-learn torch --index-url https://download.pytorch.org/whl/cpu
```

(Later you’ll use CUDA / Colab for GPU.)

## 6. How to take notes (simple system)

For every session, answer in `my-notes/`:

1. **What did I learn in one sentence?**  
2. **What confused me?**  
3. **What did I build or try?**  
4. **One analogy** I’d use to explain this to a church/ops leader  
5. **Next session goal**

That’s enough. Fancy notes systems fail; this one sticks.
