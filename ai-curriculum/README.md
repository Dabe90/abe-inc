## How to use this (2 minutes)

1. Prefer the live app: **[Abe Stack University](../university/)** (progress, labs, flashcards).
2. Or open **[WEEK-01-CALENDAR.md](WEEK-01-CALENDAR.md)** + **[00-start-here.md](00-start-here.md)**.
3. Follow **[sessions/](sessions/)** week files (Session A → B → C) — all 120 plans are ready.
4. After each session: copy `my-notes/TEMPLATE-session.md` (example: `my-notes/2026-07-22-session-01.md`).
5. Bookmark **[free-resources-master-list.md](free-resources-master-list.md)**.
