# Firebase + Static Site Form Starter

Minimal pattern for delivering contact or enrollment forms to an inbox without a backend server.

## What's included

- `contact-form.html` — accessible form with honeypot spam field
- Web3Forms integration (swap for FormSubmit or your API)
- Notes for Firebase Hosting deploy

## Quick start

1. Sign up at [web3forms.com](https://web3forms.com) and get an access key
2. Replace `YOUR_ACCESS_KEY` in `contact-form.html`
3. Deploy to Vercel, Firebase Hosting, or any static host
