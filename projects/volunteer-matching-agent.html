Problem

Volunteer coordinators spent hours each week matching skills and availability to open shifts manually. Spreadsheet exports and gut-feel assignments didn't scale as signups passed 500 in the first month.

Solution

  • Genkit-style agent reads Firestore roster and shift availability in real time
  • Proposes ranked shift matches with reasoning the coordinator can review
  • One-click approve / edit / reject before any email goes out
  • Daily digest drafts generated by the agent, sent via Gmail API after approval
  • Audit log of every agent recommendation and human decision

Results

Leadership stopped hand-matching volunteers to shifts. Coordinators review agent proposals in minutes instead of rebuilding rosters from scratch. Manual matching work dropped an estimated 70% through peak serve season — with zero unsupervised outbound emails.
